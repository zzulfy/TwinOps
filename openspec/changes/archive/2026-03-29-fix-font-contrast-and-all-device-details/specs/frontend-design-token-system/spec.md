## MODIFIED Requirements

### Requirement: High contrast text tokens
The frontend token system SHALL provide primary and secondary text color tokens that maintain high readable contrast on white backgrounds across dashboard, widgets, and detail pages.

#### Scenario: Text remains clear on white shell surfaces
- **WHEN** a user views labels, values, and metadata in white-theme panels
- **THEN** primary text SHALL remain clearly distinguishable from the background
- **AND** secondary text SHALL preserve hierarchy without appearing faded or hard to read
